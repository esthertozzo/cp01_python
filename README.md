# 🟢 Matrix Terminal - Chatbot Interativo

Projeto desenvolvido para o **Checkpoint 1** da disciplina de *Computational Thinking with Python*.

---

## 🎬 Sobre o Projeto

Este programa simula um terminal interativo inspirado no universo do filme *Matrix*, criando uma experiência imersiva para o usuário.

Ao iniciar, o sistema apresenta uma sequência de inicialização com mensagens temáticas (como “Você tomou a pílula vermelha...”), simulando o acesso à Matrix.

O usuário é identificado como um **operador** e pode interagir com um menu de opções no terminal.

---

## ⚙️ Funcionalidades

- 📅 Exibir data atual do sistema  
- ⏰ Exibir hora atual  
- 🧠 Gerenciar lista de dados:
  - Adicionar itens
  - Listar itens
  - Contar itens cadastrados  
- 🔮 Exibir mensagens temáticas do “Oráculo”  
- 💻 Interface estilizada com cores e efeito de digitação  

---

## 🧠 Conceitos Utilizados

- Estruturas condicionais (`if/elif`)
- Laços de repetição (`while`)
- Listas
- Funções
- Manipulação de data e hora (`datetime`)
- Biblioteca `rich` para estilização do terminal

---
